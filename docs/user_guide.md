# User Guide

## Overview

LiVision is a fast lidar simulation library for multiple robots. This guide explains how to build and run the example, then points to the API reference.

## Dependencies

- bgfx.cmake
- SDL2
- imgui
- implot

## Run Example

```bash
./setup.sh
./build.sh
./build/example/livision_example

## Build & install

```bash
./setup.sh
./build.sh
./install.sh
```
