# User Guide

## Overview

LiVisionは複数ロボット向けの高速LiDARシミュレーションライブラリです。このガイドではビルドとサンプル実行を説明します。

## Dependencies

- bgfx.cmake
- SDL2
- imgui
- implot

## Run Example

```bash
./setup.sh
./build.sh
./build/example/livision_example
```

## Build & install

```bash
./setup.sh
./build.sh
./install.sh
```
